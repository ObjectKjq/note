## 媒体查询

```css
#div0{
    width: 100px;
    height: 200px;
}
/*
当尺寸为什么时就展示什么样式
device-width:设备屏幕的宽度
width：浏览器的宽度
*/
@media /*媒体查询*/ screen /*表示屏幕*/ and (min-device-width: 200px) and (min-device-width: 100px){
    #div0{
        
    }
}
```
